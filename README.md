- # 项目提案：植灵养成师（移动端 / 小程序）—— 完整版

  ## 1. 项目概述

  **产品名称**：植灵养成师

  **一句话简介**：一款以**现实植物养护 + AI 识别 + 虚拟精灵养成**为核心的虚实融合养成游戏，让用户在养护真实植物的过程中，同步养成专属精灵，在游戏化乐趣中学习园林知识、培养责任感。

  **目标用户**：学生、年轻职场人、植物爱好者、喜欢养成游戏的用户、希望接触自然但缺乏养护知识的人群。

  **核心理念**：现实养花驱动虚拟成长，AI 识别连接虚实，用游戏化让植物养护变轻松、有趣、有成就感。

  ## 2. 核心功能模块

  ### 2.1 植物绑定与 AI 识别

  - 支持拍摄真实植物，AI 自动识别品种、科属、生长习性。
  - 自动匹配该植物的**园林标准生长周期**：幼苗期→青年期→壮年期→弱株期。
  - 每日拍照打卡，AI 判断植物健康状态、长势、是否缺水 / 缺肥 / 生病。
  - 生长连贯性校验，防止网图作弊，保证真实性。

  ### 2.2 植灵成长系统（核心创新）

  - 每株植物对应一只专属**植灵**，形态随植物生长阶段变化。
  - 壮年期植灵最强，解锁全部技能与外观；弱株期属性下降，无法战斗。
  - 经验值来自：有效打卡、健康状态、连续养护、治愈病害。
  - 多年生植物可长期保持壮年期，一二年生植物支持**种子传承**，保留前代强度。

  ### 2.3 养护与科普系统

  - 根据植物品种，自动推送**专业园林养护方案**：浇水、光照、施肥、修剪、温湿度。
  - 植物异常时，AI 识别病害并给出救治方法。
  - 每养成一种植物，解锁完整图鉴：学名、习性、养护日历、病虫害防治。
  - 新手保护期：7 天内不惩罚、只引导，降低入门压力。

  ### 2.4 植灵对战与互动

  - 壮年期植灵可参与轻量级对战，技能与植物特性强绑定：

    - 多肉 → 防御肉盾
    - 月季 → 高爆发输出
    - 绿萝 → 辅助回血
    - 薄荷 → 控制减益

    

  - 对战胜利获得成长道具、外观装饰、稀有图鉴。

  ### 2.5 生命周期与容错机制

  - 不搞强制死亡：植物死亡后植灵进入**休眠**，重新种植即可唤醒继承进度。
  - 断签、养护失误不惩罚，只给出改善建议。
  - 种子传承：自然枯萎的植物可留下种子，新植灵继承部分属性，形成养成循环。

  ### 2.6 图鉴与成就系统

  - 全植物图鉴收集，包含专业园林知识 + 精灵全阶段立绘。
  - 成就：养护大师、救治达人、多年生守护者、图鉴收藏家。
  - 支持将精灵与植物合照分享，形成传播点。

  ## 3. 用户体验流程

  1. 首次打开：选择 / 拍摄自己的真实植物 → AI 识别 → 生成专属植灵。
  2. 日常打卡：给植物浇水 / 拍照 → AI 判定健康 → 植灵获得成长。
  3. 阶段进阶：植物进入青年 / 壮年期 → 植灵进化、解锁新形态新技能。
  4. 健康预警：植物生病 → AI 提示 → 按方案救治 → 植灵恢复并奖励。
  5. 长期养成：收集多种植物 → 组建植灵队伍 → 对战 / 收集 / 分享。

  ## 5. 数据模型（核心）

  **用户表**

  userId、昵称、头像、经验、等级、成就、设置

  **植物表**

  plantId、userId、品种名称、科属、当前周期、健康值、累计养护天数、最后打卡时间

  **植灵表**

  spiritId、plantId、形态阶段、属性、技能、外观、亲密度、战力

  **打卡记录表**

  recordId、plantId、打卡时间、AI 健康结果、图片、获得奖励

  **图鉴表**

  bookId、植物名称、习性、养护知识、解锁状态

  ## 6. 亮点与差异化

  - **真正虚实绑定**：不是虚拟养成，是**现实植物状态直接决定精灵强度**。
  - **AI + 园林专业**：科学周期、健康判断、科普知识，比普通游戏更专业。
  - **无惩罚正向体验**：不死亡、不清零、不劝退，温柔引导长期坚持。
  - **教育 + 娱乐 + 实践**：玩游戏 = 学养护 = 养真植物，三重价值。
  - **低门槛高沉浸**：不用懂园林，拍照就能玩，适合所有新手。

  ## 7. 商业变现（不影响体验）

  - 精灵外观、装饰、场景皮肤。
  - 高级图鉴、稀有植物主题包。
  - 去广告、云端备份、多植物同时托管。
  - 植物周边、花肥、花盆电商导流（温和不强制）。

  ## 8. 隐私与安全

  - 拍照仅用于 AI 识别，不上传无关隐私。
  - 本地优先处理，隐私安全可控。
  - 用户可随时删除数据、注销账号。
  - 符合青少年模式与个人信息保护规范。

  ## 9. 开发阶段规划

  **MVP（1.5 个月）**

  植物识别、基础打卡、植灵成长、周期变化

  **迭代 1（1 个月）**

  健康判断、病害提示、养护知识推送、基础图鉴

  **迭代 2（1.5 个月）**

  植灵对战、传承机制、防作弊、分享功能

  **迭代 3（1 个月）**

  成就系统、UI 优化、全流程体验打磨

  植灵养成师是一款**创新、落地、有教育意义、强趣味性**的虚实融合养成游戏。
